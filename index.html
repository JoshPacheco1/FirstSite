  • Synthesizers I used:
  • ES1 which uses subtractive synthesis
  • ES2 which combines subtractive synthesis with elements of FM and wavetable synthesis
  • EFM1 is a powerful frequency modulation synthesizer
  • RetroSynth is a 16-voice synthesizer
  • ES M is a monophonic synthesizer used for bass sounds that punch
  • Ultrabeat is a synthesizer used to create percussive sounds and polyphonic rhythms
